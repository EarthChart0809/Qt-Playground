## ドット絵作成ツール

このプロジェクトは、PySide6 を使用して作成されたドット絵作成ツールです。ユーザーは、キャンバス上でドット絵を描画し、レイヤーを管理することができます。


## 特徴

- ドット絵の描画
- レイヤーの追加、削除、順序変更
- レイヤーの透明度設定
- グリッド表示のオン/オフ
- 画像の読み込みと保存


## 説明

このドット絵作成ツールでは、レイヤーやグリッドなどの基本的な機能が搭載されています。ユーザーは、複数のレイヤーを追加して、それぞれのレイヤーに異なるドット絵を描画することができます。また、レイヤーの順序を変更したり、透明度を設定することも可能です。さらに、グリッド表示をオン/オフすることで、ドット絵の描画をより正確に行うことができます。

このツールでは、OpenCV と NumPy を使用して画像の読み込みと保存を行います。特に、Lab空間を用いて画像の減色加工を行い、読み込んだ画像をドット絵に変換する機能も備えています。これにより、ユーザーは任意の画像をドット絵として再現することができます。


## インストール

このプロジェクトを実行するには、以下の依存関係をインストールする必要があります。`requirements.txt` ファイルを使用して、必要なパッケージをインストールします。


## ファイル構成

main.py: アプリケーションのエントリーポイント
DotEditor.py: ドット絵エディタのメインウィジェット
LayerSetting.py: レイヤー管理のウィジェット
CropSelection.py: 画像の選択範囲を管理するウィジェット
requirements.txt: プロジェクトの依存関係
